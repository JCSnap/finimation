# CAPM & Security Market Line

The Capital Asset Pricing Model links expected return to systematic risk (beta).

## CAPM Equation

$$E[R_i] = R_f + \beta_i\big(E[R_m]-R_f\big)$$

The **Security Market Line (SML)** is the graph of this equation.

## Alpha

Given actual expected return $R_i^{obs}$:

$$\alpha_i = R_i^{obs} - E[R_i]$$

- Positive alpha: above SML (outperforming CAPM-implied return)
- Negative alpha: below SML
